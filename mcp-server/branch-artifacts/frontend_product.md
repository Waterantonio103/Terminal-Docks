## Product Decisions
- Build as a single-page static browser app using `index.html`, `styles.css`, and `app.js`.
- First viewport must clearly name Harbor Grid and position it as solar, battery, and outage-readiness planning for waterfront buildings.
- Include a clear conversion path with hero and final CTAs.
- Include proof content around site survey, load mapping, and phased roadmap.
- Include at least one keyboard-accessible interaction that updates visible readiness guidance.
- Avoid real energy savings claims, compliance promises, analytics, cookies, third-party embeds, and backend lead capture.
